# **App Name**: TalentVerse

## Core Features:

- Professional Showcase: Display a grid of professionals with key information like name, role, skills, and satisfaction rate.
- Filtering & Sorting: Allow users to filter professionals by category and search by name, role, or skills.  Include a 'reshuffle' option to randomize the order of the results.
- Professional Details Modal: On selecting a professional, display detailed information, including a larger profile picture, full skill list, experience, and contact information in a modal.
- AI Skill Match: Employ an AI tool to analyze user's project requirements and automatically suggest professionals with the best-matched skills and experience.

## Style Guidelines:

- Primary color: A deep indigo (#554686) to convey professionalism and trust.
- Background color: A light gray (#F4F3F6) to ensure readability and a clean interface.
- Accent color: A vibrant purple (#A770EF) for interactive elements and highlights to draw the user's eye.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface. Suitable for both headlines and body text.
- Use simple, outline-style icons from Lucide to maintain a clean and professional look.
- Use a grid-based layout for the professional listings to provide a structured and easy-to-browse experience.
- Subtle fade-in animations for loading content and smooth transitions for modal appearances to enhance user experience.
